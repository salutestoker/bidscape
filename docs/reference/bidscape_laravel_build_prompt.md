# Bidscape Reference Notes

This project follows the addendum-integrated V1 workflow:

- Lead intake and qualification.
- Estimate building with assemblies, catalog components, and custom scope lines.
- Branded estimate PDF generation and email delivery.
- Public customer review link for approve/sign or decline.
- Customer, signed contract record, sold job, pending deposit, and immutable job packet creation after signature.
- Internal deposit recording.
- Sales reports for summary, lead source performance, and estimate conversion.

Bidscape V1 is sales-only. Do not add scheduling, field execution, crew management, payroll, inventory consumption, actual-cost tracking, or production status workflows.
